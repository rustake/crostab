Crostab and table for cross-tabular analysis

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
crostab = ">=0.0.1"
```

and this to your crate root:

```rust
extern crate crostab;
```

## Compatibility

The `crostab` crate is tested for rustc 1.48 and greater.
